| for               | false    | String          | Client identifier of the component for which to display messages. |

| infoClass         | false    | String          | CSS style class to apply to any message with a severity class of "INFO". |

| warnClass         | false    | String          | CSS style class to apply to any message with a severity class of "WARN". |
